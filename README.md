# moveit

This is a Pomodoro timer application built with NextJS. The main goal of moveit is to remember you that you need to exercise while coding in a fun way, with challenges that give you experience while leveling up your account! ü•á

## How to use?

- Open the application and start the timer
- After the 25 minutes, a challenge will appear for you to move yourself
- After finishing the challenge, confirm that in the application and gain experience points
- Keep that to up your level and keep healthier! Ô∏èÔ∏èüèã

## Technologies

moveit was built using:

- [React](https://reactjs.org)
- [NextJS](https://nextjs.org)

## Installation

Install the dependencies and start the server.

```sh
cd moveit
npm i
npm dev
```

or...

```sh
cd moveit
yarn install
yarn dev
```
